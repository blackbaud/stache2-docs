Stache uses the SKY UX localization feature that requires SKY UX Builder 1.14.0 or higher. To update the SKY UX Builder dependency, open the package.json file in your project's root directory, and under the devDependencies property, update @blackbaud/skyux-builder to the latest version in the SKY UX Builder changelog .

In addition, Stache users must include a resources file in their project to render the components and their text correctly until a fix is available in SKY UX. To include the resource file, create a locales folder in your project's src/assets folder, and then add a resources_en_US.json file to that folder. You can specify localization strings in the resources_en_US.json file to reuse throughout your SPA, but to render components, you just need to make sure the file is in place.
