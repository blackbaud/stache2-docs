Stache is a component library designed to work with SKY UX single-page applications (SPAs) for the purpose of creating elegant and helpful technical documentation websites. From this page, you'll learn what's required for set up, how to get Stache working with your SPA, and recommended next steps. Let's get started. Install prerequisites

To prepare your machine for your new {{ stache.jsonData.global.productNameShort }} documentation site, you need to install SKY UX's prerequisites.

Complete the SKY UX Initial setup instructions .

Set up a Stache SPA

Depending on your situation, you can set up a Stache site in one of these ways.

Blackbaud RDO staff

We provide specific documentation for Blackbaud RDO staff, because you have additional features when setting up a new Stache SPA. We provide a template and set you up with an Azure DevOps Git repo with a build and release pipeline! If you are a Blackbaud staff member but don't have access to the RDO Azure DevOps instance, then follow the Set up a Stache SPA steps below.
