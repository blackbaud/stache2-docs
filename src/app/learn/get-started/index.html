Prerequisites

Stache 2 components require SKY UX 2. For information about how to install SKY UX 2 and its prerequisites, see the SKY UX setup instructions.

Create a Stache application

After the SKY UX CLI and its prerequisites are in place, creating a Stache application is quick and easy. You just specify the Stache 2 template when you create your SKY UX application.

  1. From the command prompt, cd into the directory where you want to create the project.

  2. Run the skyux new command use the --template option to specify the Stache template: skyux new --template stache. This creates a SKY UX application with the Stache template and components.

    For more information about how to create a SKY UX project, see the SKY UX instructions.

Add Stache to an existing SKY UX application

To add the Stache template and components to an existing SKY UX project, follow our tutorial to add Stache.
