Stache is a component library designed to work with SKY UX single-page applications (SPAs) for the purpose of creating elegant and helpful technical documentation websites. From this page, you'll learn what's required for set up, how to get Stache working with your SPA, and recommended next steps. Let's get started. Install prerequisites

To prepare your machine for your new {{ stache.jsonData.global.productNameShort }} documentation site, you need to install SKY UX's prerequisites.

  1. To load your local single-page application into SKY UX Host, you must install the skyux-ca.crt root certificate. Follow the SSL certification installation instructions on the SKY UX website.

  2. Complete the SKY UX Initial setup instructions .

    3. Blackbaud staff must also ensure they have access to Visual Studio Team Services (Azure DevOps) and link their Azure DevOps credentials from the Self-contained systems page.
