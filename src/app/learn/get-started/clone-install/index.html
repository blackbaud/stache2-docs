Clone and Install a {{ stache.jsonData.global.productNameShort }} SPA

This tutorial provides step-by-step instructions on how to clone an existing Stache SPA from VSTS and install locally.

Before you begin this tutorial, you need to complete the SKY UX prerequisites. Clone the Git Repo from VSTS

  1. From VSTS Git, select the repo name field and then select the All repositories tab. Use the search to find your repo. Almost all Stache SPAs start with skyux-spa.

    If you're still unsure if you have the correct VSTS repo, ask the person who set up the Stache SPA for you.

  2. Select the Clone button.

  3. Under Command line, make sure the "HTTPS" button is selected and select the copy button.

  4. Open a terminal or command prompt on your local machine where you store all of your Git repos. Enter git clone and then paste the link you just copied.
  5. Press Enter (Windows) or Return (Mac). The repo clones locally.
NPM Install

After the repo finishes cloning, change your directory into the new repo folder. For example, cd skyux-spa-engineering-system-docs.

To be able to serve locally, you need to install the SKY UX dependencies. In the command prompt, enter npm install.

This will take a few minutes. Once it completes, you can test the new install by serving locally. Enter skyux serve. When it finishes preparing to serve, a browser will open to a local development instance of your Stache SPA.
