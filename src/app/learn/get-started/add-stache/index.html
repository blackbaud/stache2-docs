Add {{ stache.jsonData.global.productNameShort }} to a SKY UX Application This tutorial provides step-by-step instructions to add {{ stache.jsonData.global.productNameShort }} to an existing SPA. Install dependencies npm install --save @blackbaud/skyux-lib-stache npm install --save @blackbaud/skyux-builder-plugin-stache

If you plan to use the Blackbaud Identity Service authentication, install the email whitelist plugin.

npm install --save @blackbaud/skyux-builder-plugin-auth-email-whitelist
Add the {{ stache.jsonData.global.productNameShort }} module to your application

Open the app-extras.module.ts file in your project. Export StacheModule to make {{ stache.jsonData.global.productNameShort }} components available throughout the project.

Declare plugins

Open the skyuxconfig.json file in your project. Add the required {{ stache.jsonData.global.productNameShort }} plugins to the plugins array.
