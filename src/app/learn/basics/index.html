Navigation

{{ stache.jsonData.global.productNameShort }} provides several navigational utilities for moving users quickly thorough your content. Most of these are enabled by default and handled for you automatically.

Back to Top Set by default

For long form content, having to scroll manually to the top of the page is annoying. The back to top button provides a quick way to navigate back to the top of a page. Back to top is enabled by default. You can disable it on the stache tag.

To use local images in your {{ stache.jsonData.global.productNameShort }} project, add them to your src/assets directory in the source folder. You can then reference them in your {{ stache.jsonData.global.productNameShort }} pages like so:

<stache-image imageSource="~/assets/hero-sample-1.jpg"> </stache-image>

Every image must be referenced with the ~/assets​/ prefix for SKY UX to register it as a static asset and render it.
