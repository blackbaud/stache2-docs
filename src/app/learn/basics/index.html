As you are getting to know Stache, use the documentation below to understand how it's organized and how to create documentation using Stache. For information about specific components, refer to our Components. Stache Wrapper

The Stache Wrapper is the primary tag in a Stache project, and contains all the other Stache tags and content. Most of your page level configuration is done through the Stache Wrapper, including layouts and navigation. Typically, this is the first tag you will want to add to any Stache page.

Stache takes a prescriptive approach to configuration. Most of the properties on the Stache tag have defaults, so you will rarely have to set them yourself. The main exception is the page title.
