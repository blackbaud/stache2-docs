Prerequisites

Stache components require SKY UX to use. To learn more, refer to the SKY UX documentation.

Start a Stache Project

Starting a Stache project is quick and easy. To get started, from the command prompt, cd into the directory where you want to create the project and run skyux new --template stache. This will generate a new SKY UX project with the Stache template and components. For more information about starting a new SKY UX project, refer to their documentation.

Add Stache to an Existing SKY UX Project

If you want to add Stache to an established project:

  1. Run the following command in your project directory: npm install --save @blackbaud/stache npm install --save @blackbaud/skyux-builder-plugin-stache-code-block npm install --save @blackbaud/skyux-builder-plugin-stache-includes
  2. Add the StacheModule to your app's app-extras.module.ts. Import the module and add it to the imports and exports to use the Stache components throughout your project.
  3. Import the `SkyAppConfig` and `StacheConfigService` into app-extras.module.ts and add them to the `resolves` array. import { NgModule } from '@angular/core'; import { SkyAppConfig } from '@blackbaud/skyux-builder/runtime'; import { StacheModule, StacheConfigService } from '@blackbaud/stache'; // Specify entry components, module-level providers, etc. here. @NgModule({ imports: [ StacheModule ], exports: [ StacheModule ], providers: [ { provide: StacheConfigService, useExisting: SkyAppConfig } ], entryComponents: [] }) export class AppExtrasModule { }
