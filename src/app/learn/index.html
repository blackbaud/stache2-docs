Prerequisites

Stache components require SKY UX to use. To learn more, refer to the SKY UX documentation.

Start a Stache Project

Starting a Stache project is quick and easy. To get started, run skyux new --template stache. This will generate a new SKY UX project with the Stache template. Once the project generates, run npm install to install the necessary dependencies.

Add Stache to an Existing SKY UX Project

If you want to add Stache to an established project:

  1. Run npm install --save @blackbaud/stache from your project directory.
  2. Add the StacheModule to your app's app-extras.module.ts:
import { NgModule } from '@angular/core'; import { StacheModule } from '@blackbaud/stache'; // Specify entry components, module-level providers, etc. here. @NgModule({ imports: [ StacheModule ], exports: [ StacheModule ], entryComponents: [] }) export class AppExtrasModule { }
