The code block component allows you to format a block of text as a code snippet with syntax highlighting. You wrap stache-code-block tags around the code block.

To format shorter code snippets that appear inline within a larger block of text, use the code component instead.

Code block properties

  • code — (Optional) Passes a code sample into the component.
  • languageType — (Optional) Indicates the language of the code sample. Supports a variety of languages. The default language is markup.
