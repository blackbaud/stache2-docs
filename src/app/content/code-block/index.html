The code block component allows you to format a block of text as a code snippet with syntax highlighting by wrapping stache-code-block tags around the code block. The code block component is equipped with copy-to-clipboard functionality. When users hover over the code block area, a copy-to-clipboard button appears, and the formatted code copies with the formatting intact. To format shorter code snippets within larger blocks of text, use the code component instead.

If a code block has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, you must use "&lt;" to escape < and "&gt;" to escape >. For example, you must escape the < and > characters in the code snippet public Task<Generic> MethodName() or else the component automatically appends </Generic> to the end: public Task MethodName().

Code block properties

  • code — (Optional) Passes a code sample into the component.
  • languageType — (Optional) Indicates the language of the code sample. Supports a variety of languages . The default language is markup.
