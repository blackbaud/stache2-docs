The footer enables you to add useful links and details, like copyright information, to the bottom of your site.

Footer properties

  • copyrightLabel — Specifies the text to display in the footer's copyright information. By default, the footer displays a © symbol and the current year followed by "Blackbaud, Inc. All rights reserved." This property enables you to replace the default text that appears after the year with your own copyright text.

  • nav — This flag contains an items array, enabling you to include specific links in the footer.

  • siteName — Value read from the skyux.app.title attribute in your skyuxconfig.json file.

To enable the footer globally across your SPA, add footer to your skyuxconfig.json file under "appSettings": "stache":. To disable the footer on a specific page, add the showFooter="false" attribute to the Stache wrapper.

Demo

See the bottom of this page for a demo of the footer.

Code
"app": { "title": "Blackbaud Stache Documentation" }, "appSettings": { "stache": { "footer": { "copyrightLabel": "My Company, Inc.", "nav": { "items": [ { "title": "Privacy Policy", "route": "/privacy-policy" }, { "title": "Terms of Use", "route": "/terms-of-use" } ] } } } } <stache showFooter="false"> </stache>
