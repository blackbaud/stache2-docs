The code component allows you to format text as a code snippet within a larger block of text. Within the block of text, you wrap stache-code tags around the code snippet.

To format longer code snippets that stand alone as separate blocks of text, use the code block component instead.

The code component formats code snippets such as stache-code and escapeCharacters.

When you set the escapeCharacters property to true, the component renders special characters as text. For example, you can instruct users to "add span tag content."

When you do not set the escapeCharacters property to true, the component renders special characters as code. For example, the same instruction as in the previous bullet renders as "add span tag content."
