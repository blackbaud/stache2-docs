The edit button facilitates contributions by adding a link to the repository on each page in your site. This feature is enabled globally, but you can opt out on individual pages.

Edit button properties

  • showEditButton â€” When you enable the edit button for the entire site in appSettings, all pages will then display the edit button. To disable the edit button for a single page, add this flag to the Stache wrapper on that page and set it to false.

Enable the edit button

The edit button is not enabled by default. To enable the edit button, add editButton to your skyuxconfig.json file under "appSettings": "stache":.

"appSettings": { "stache": { "editButton": { "url": "URL to your repository", "text": "The text for your edit button. The default is 'Edit'" } } } Demo

"appSettings": { "stache": { "editButton": { "url": "https://github.com/blackbaud/stache2-docs" } } } <stache showEditButton="false"> </stache>
