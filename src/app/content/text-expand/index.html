This SKY UX component truncates long text with an ellipsis and a link that allows the user to fully expand the text. If the text length falls below the specified threshold, then no action is taken.

Text expand properties

  • text — The text to truncate.
  • maxLength — (Optional) The number of characters to show before truncating the text. To avoid truncating text in the middle of a word, the directive looks for a space in the 10 characters before the last character (default: 200).
  • maxExpandedLength — (Optional) The maximum number of characters to show in the inline expanded view. If the text includes more characters, then a modal view with the content will be displayed when the 'See more' link is clicked (default: 600).
  • expandModalTitle — (Optional) The title to display in the modal expanded view (default: Expanded View).

text expand repeater properties

  • data — The data to truncate.
  • maxItems — The number of items from the data to display before truncating.
