The image component provides a responsive container to display an image. To display images on your {{ stache.jsonData.global.productNameShort }} site, add them to the src/assets directory in the source folder.

  • src
    • assets
      • image-name.jpg

If your Stache site requires a lot of image files, we recommend you organize the images into subfolders by page name.

After the images are in place, you can reference them in your {{ stache.jsonData.global.productNameShort }} pages. You must reference every image with the ~/assets​/ prefix in order for SKY UX to register it as a static asset and render it.

Image properties

  • imageSource — The path to the image to render.
  • imageAlt — The alt tag to describe the image for accessibility.
Demo
Code
<stache-image imageAlt="This is an example hero." imageSource="~/assets/image-name.jpg"> </stache-image>
