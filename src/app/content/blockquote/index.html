The blockquote component provides a stylized and formatted blockquote.

Blockquote properties

  • author - (Optional) Sets the source author / quotee.
  • quoteSource - (Optional) Sets the link to the source of the quote.
If you set the quoteSource, we highly recommend you set the author. Otherwise the default will display as — Source. Demo
