The stache-row and stache-column components are deprecated as of Stache v2.15.0 and may be unavailable in Stache v3.0.0. Use the SKY UX fluid grid component instead.

Stache grid docs - deprecated

The grid components provide a 12 column layout grid for organizing content for all device sizes. This enables you to lay out content in columns that know how to be responsive based on screen size. For example, you can have content that has 3 columns across on a large screen, but collapses into a single column on a small (ex. mobile) screen. The grid component is not recommended as a table replacement.

The grid consists of stache-row and stache-column tags. Every column must be contained in a row. We recommend you use this component to arrange content to maximize the width of a page, but be flexible for different screen sizes.

{{ stache.jsonData.global.productNameShort }} Row properties

  • reverseColumnOrder — (Optional) Reverses the display order for the columns when set to true.

{{ stache.jsonData.global.productNameShort }} Column properties

  • screenSmall — The number of columns (1-12) for a small screen (768px).
  • screenMedium — The number of columns (1-12) for a medium screen (992px).
  • screenLarge — The number of columns (1-12) for a large screen (1200px).

To handle behavior for multiple screen sizes, you can combine more than one size for each column. For example: <stache-column screenSmall="8" screenMedium="9" screenLarge="10">. This is helpful if you want to more carefully control the breakpoints.
