In Angular, relative links for navigation within a site are implemented differently than external HMTL href links. To simplify the process, we created a Stache link directive that works for external links and internal navigation within a Stache site, including links to anchor points on the same page.

Folder names make up the file path, and because links are case sensitive, changes in casing will cause a broken link. When naming your folders, keep the folder names short and use lowercase. Do not use spaces or underscores but use hyphens instead.

Link properties

  • stacheRouterLink — Specifies the link path. To link to a section on the same page, pass in . for the route instead of the full path.

  • fragment — (Optional) Specifies the ID of the anchor point. For Stache page anchors, the ID is a hypenated, lowercase version of the text.
