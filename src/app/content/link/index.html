The link directive allows for navigation within a Stache site, including anchor points on a page.

Link properties

  • stacheRouterLink — Specifies the path to link to. To link to a section on the same page, pass in . for the route instead of the full path.

  • fragment (Optional) — Specifies the ID of the anchor point. For Stache page anchors, the ID will be a hypenated, lowercase version of the text.
