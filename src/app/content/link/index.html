The link directive allows for navigation within a Stache site, including anchor points on a page.

Link properties

  • stacheRouterLink — The path to link to. If you are linking to a section of the same page, you can pass . in for the route rather than the full path.

  • fragment (Optional) — The Id of the anchor point. If using Stache Page Anchors, the Id will be a hypenated, lower case version of the text.
