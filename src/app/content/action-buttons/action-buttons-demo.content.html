The Action Buttons component provides a clean, visual navigation pattern for linking to multiple documents using the SKY UX Action Button.

Action Buttons properties

  • routes — Specifies the properties for each action button in the list. Each button has the following properties:

    • name — The name of the button.
    • path — The path the button will route to when clicked.
    • icon — The font awesome icon for the button.
    • summary — A description for the button.
Demo
Code
import { Component } from '@angular/core'; @Component({ selector: 'stache-action-buttons-demo', templateUrl: 'action-buttons-demo.component.html' }) export class StacheActionButtonsDemoComponent { public routes = [ { name: 'Layout', path: '/components/layout', icon: 'map-o', summary: 'Basic page layouts using Stache and skyux pattern, for quick page layouts.' }, { name: 'Layout: Sidebar', path: '/components/layout-sidebar', icon: 'columns', summary: 'Side bar navigation layout pattern for creating quick documentation.' } ]; }
