The copy-to-clipboard component allows users to copy content to their clipboard. The content to be copied must be wrapped in an element that can be referenced with the #identifier notation. A stache-copy-to-clipboard element must then be added and contain a [copyTarget] attribute that references the identifier from the previously created element. You can optionally include a buttonText and buttonClickedText attribute to alter the button text. When the button is clicked, all text content within the copyTarget element is copied to the user's clipboard with formatting intact. The content can then be pasted in to any field that can receive text.

Only HTML elements can use the #identifier notation. If you would like to copy the content of a non-HTML element, such as a Stache component, that element must be wrapped in a div or another valid HTML element.

Copy to clipboard properties

  • copyTarget — Section of content to be copied.
  • buttonText — (Optional) Text to be displayed by copy-to-clipboard button.
  • buttonClickedText — (Optional) Text to be displayed by copy-to-clipboard button once it has been clicked.
