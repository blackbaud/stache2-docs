The affix component is an Angular 2 component that causes the affected components to affix to the top of your browser window as you scroll past it on the page. It makes use of the Affix Top directive, to add styles and the stache-affix-top class to your wrapped element or component.

The component is the recommended way to use the affix top feature. When an element affixes to the top of the window, an empty place holder element is left behind, maintaining the pages size ratios.
