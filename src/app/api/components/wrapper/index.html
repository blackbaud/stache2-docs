The wrapper component is the primary component for {{ stache.jsonData.global.productNameShort }}. It handles configuration, layouts, and routing for every {{ stache.jsonData.global.productNameShort }} page.

Wrapper properties

  • pageTitle — Sets the heading for the page.
  • layoutType — (Optional.) You can specify a specific type of layout for each {{ stache.jsonData.global.productNameShort }} page. The default is sidebar.
    • sidebar — Displays a sidebar navigation on the page for each child directory.
    • container — Displays content in a responsive container with margins on either side.
    • blank — Displays content with no container.
  • showBackToTop — Determines if the back to top button displays on the page when the user scrolls past a 200px offset from the top. The default is true.
  • showBreadcrumbs — Determines if the breadcrumb navigation displays on the page. The default is true.
  • showTableOfContents — Determines if the table of contents displays on the page. The default is false.
  • sidebarRoutes — The routes used to populate the sidebar navigation. These are set automatically by {{ stache.jsonData.global.productNameShort }}.
  • inPageRoutes — The routes used to populate the table of contents navigation. These are set automatically by {{ stache.jsonData.global.productNameShort }}.
  • breadcrumbsRoutes — The routes used to populate the breadcrumbs navigation. These are set automatically by {{ stache.jsonData.global.productNameShort }}.
  • navOrder — This number dictates the order in which your routes will render in the sidebar next to their sibling routes.
