The Markdown component allows users to write Markdown in their HTML files. Within the stache-markdown tags, users can mix Markdown and HTML elements to create documentation with Markdown without losing out on {{ stache.jsonData.global.productNameLong }} features.

Stache-specific Markdown

  • Page anchors — The ## syntax creates an h2 heading with a Stache page anchor around it and adds the heading to the table of contents.

    To create a regular h2, simply use the HTML

    tags.

  • Code blocks — The ``` syntax creates a Stache code block. You can specify a language directly after the opening ```. For example, ```js creates a stache-code-block element with the languageType attribute set to js.
