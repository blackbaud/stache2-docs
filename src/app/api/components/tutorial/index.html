The stache-tutorial-component tag is a wrapper that holds the rest of the information (heading, summary, and steps) for the tutorial on your page.

This optional tag is the title for your Turtorial This optional tag is the summary for the following tutorial

The stache-tutorial-step-component tag is used for each step in your tutorial. It has the optional property showNumber. This allows you to decide if you want your tutorial step to have a number associated with it. (defaults to true).

This optional tag is the steps title. This tag should wrap your content for the tutorial step.
