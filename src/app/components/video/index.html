    YouTube

    To find the YouTube URL to use as a video source, grab the source URL from the embed code. From a video's page on Youtube, select Share and then the Embed tab. Copy the src="" URL to use with the video component.

    Vimeo

    To find the Vimeo URL to use as the video source, grab the source URL from the embed code. From a video's page on Vimeo, select the share button and then copy the embed code. Vimeo does not enable you to select a portion. Paste the code into a text editor and copy the portion of the URL you need, starting with src="".

    O365

    Blackbaud staff can host videos used on Stache sites by uploading the videos to the O365 Stache channel. If you can't access the page, request access through a Service Now ticket.

Stache SPAs

To enable more flexibility with the video component, we contributed it to SKY UX as a standalone module. This requires only a few small changes for Stache SPAs.

  • Update your version of Stache to 3.0.0 or higher. Learn how with the Update Stache documentation.
Non-Stache SPAs

Non-Stache users can use this module without Stache. Learn how to set up this module as a dependency in your SKY UX SPA project.

  1. Install the package and plugin as part of your SPAs dependencies.

    npm install @blackbaud/skyux-lib-media --save

  2. Add the module to your app-extras.module.ts file.

    import { SkyVideoModule } from '@blackbaud/skyux-lib-media'; @NgModule({ imports: [ SkyVideoModule ], exports: [ SkyVideoModule ] }) export class AppExtrasModule { }
