In Angular, if you require custom breadcrumbs or sidebar navigation, you can override the Stache breadcrumbs and sidebar navigation and provide custom, manual routing for a page or section of your SPA.

Breadcrumbs

Custom breadcrumbs can be defined directly on the html page or with a component. If you use a component, you'll want to define which breadcrumb is the active one.

  • isActive: true
  • isCurrent: true

Sidebar navigation

Custom sidebar navigation can be defined within a component.

Code
<stache pageTitle="Add a New Cosmos Database" [breadcrumbsRoutes]=" [{ path: '/help', name: 'Hub Help' }, { path: '/help/scs', name: 'SCS'}, { path: '/help/scs/details', name: 'SCS Details' }, { path: '/help/scs/details/add-cosmos', name: 'Add CosmosDb' } ]"> </stache>

Line breaks are for display purposes only.

demo.component.html file

<stache pageTitle="Demo page" [breadcrumbsRoutes]="breadcrumbs"> </stache>

demo.component.ts file

import { Component } from '@angular/core'; import { StacheNavLink } from '@blackbaud/skyux-lib-stache'; @Component({ selector: 'demo-sample', templateUrl: './demo.component.html' }) export class DemoComponent { public breadcrumbs: StacheNavLink[]; constructor () { this.breadcrumbs = [ { name: 'Home', path: 'https://developer.blackbaud.com/skyapi' }, { name: 'Documentation', path: 'https://developer.blackbaud.com/skyapi/docs' }, { name: 'Microsoft Power Platform', path: '/microsoft-power-platform' }, { name: 'Power Automate', path: '/microsoft-power-platform/power-automate', isActive: true, isCurrent: true } ]; } }

demo.component.html file

<stache pageTitle="Demo page" [sidebarRoutes]="sidebarLinks"> </stache>

demo.component.ts file
