The code block component displays a formatted block of code with syntax highlighting.

Code Block properties

  • code — (Optional.) Pass a code sample into the component. You must use the raw loader for the code to render it as a string.
  • languageType — (Optional.) The language of the code sample for syntax highlighting. You can see the supported languages here. The default is markup.
