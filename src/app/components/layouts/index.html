The sidebar navigation displays child pages for the current page. It hides all other child pages until the route is active.

The navOrder attribute on the stache tag determines the order the routes in the sidebar navigation. If navOrder is not specified or sibling routes have the same navOrder value, then the sidebar displays routes alphabetically by name.
