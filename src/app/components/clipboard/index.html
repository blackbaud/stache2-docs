The stache-clipboard component is deprecated in Stache version 2.15.0 and higher, and may be unavailable starting in Stache version 3.0.0. For Stache SPAs, learn more below about the change.

The copy to clipboard component creates a button on your page that enables users to copy the content to their computer's clipboard.

Copy to clipboard properties

  • copyTarget — Specifies content to copy.

    This property binds data, so you must use brackets to evaluate the template expression: [copyTarget]. Without the brackets, Angular treats the value as a string and initializes the target property with the string value instead of evaluating the data value.

  • buttonText — (Optional) Specifies a label for the copy to clipboard button.
  • buttonClickedText — (Optional) Specifies a label to display on the copy to clipboard button after users click it.

Add copy to clipboard to a page

  1. To use the component, wrap text in an HTML element. Example: <div>Very special text</div> tag

  2. Specify an #identifier for that element. Example: <div #awesomesauce>Very special text</div>

  3. Place a sky-copy-to-clipboard element on the page and specify the content to copy by referencing its #identifier in the copyTarget attribute. Example:

  4. When a user clicks the copy to clipboard button, the component copies the text to the user's clipboard with formatting intact. To help users know that the copy button worked, you can specify different labels for before and after the users click the button.
Stache SPAs

To enable more flexibility with the copy-to-clipboard component, we contributed it to SKY UX as a standalone module. This requires only a few small changes for Stache SPAs.

  • Update your version of Stache to 3.0.0 or higher. Learn how with the Update Stache documentation.
Non-Stache SPAs

If your SPA uses Stache version 2.15.0 - 2.18.0, this module is installed and added for you. However, non-Stache users can use this module without Stache. Learn how to set up this module as a dependency in your SKY UX SPA project.

  1. Install the package as part of your SPAs dependencies.

    npm install @blackbaud/skyux-lib-clipboard --save

  2. Add the module to your app-extras.module.ts file.

    import { SkyClipboardModule } from '@blackbaud/skyux-lib-clipboard'; @NgModule({ imports: [ SkyClipboardModule ], exports: [ SkyClipboardModule ], providers: [ ], entryComponents: [] }) export class AppExtrasModule { }
