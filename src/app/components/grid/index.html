The SKY UX fluid grid component provides a responsive 12-column layout to organize content for all device sizes.

The grid component provides a 12 column layout grid for organizing content for all device sizes. This enables you to lay out content in columns that know how to be responsive based on screen size. For example, you can have content that has 3 columns across on a large screen, but collapses into a single column on a small (ex. mobile) screen. The grid component is not recommended as a table replacement.

The grid consists of sky-fluid-grid, sky-row, and sky-column tags. Every column must be contained in a row. We recommend you use this component to arrange content to maximize the width of a page, but be flexible for different screen sizes.

Sky row properties

  • reverseColumnOrder — (Optional) Reverses the display order for the columns when set to true. Default is false.

Sky column properties

  • screenXSmall — The number of columns (1-12) for an extra small screen (less than 768px).
  • screenSmall — The number of columns (1-12) for a small screen (768 - 991px).
  • screenMedium — The number of columns (1-12) for a medium screen (992 - 1199px).
  • screenLarge — The number of columns (1-12) for a large screen (greater than 1200px).

To handle behavior for multiple screen sizes, you can combine more than one size for each column. For example: <sky-column screenSmall="8" screenMedium="9" screenLarge="10">. This is helpful if you want to more carefully control the breakpoints.
