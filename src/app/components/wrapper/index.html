The wrapper component is the primary component for {{ stache.jsonData.global.productNameShort }}. It handles configuration, layouts, and routing for every {{ stache.jsonData.global.productNameShort }} page. View this component's technical reference for more details.

Wrapper properties

  • pageTitle — Sets the heading for the page.
  • layout — (Optional) You can specify a specific type of layout for each {{ stache.jsonData.global.productNameShort }} page. The default is sidebar.
    • sidebar — Displays a sidebar navigation on the page for each child directory.
    • container — Displays content in a responsive container with margins on either side.
    • blank — Displays content with no container.
  • showBackToTop — Indicates whether to display the back to top button on the page when the user scrolls past a 200px offset from the top. The default is true.
  • showBreadcrumbs — Indicates whether to display the breadcrumb navigation on the page. The default is true.
  • showEditButton — Indicates whether to display the edit button on the page. By default, the edit button appears on all pages when you add an editButton URL to your skyuxconfig.json. To hide the edit button on a particular page, set showEditButton to false.
  • showTableOfContents — Indicates whether to display the table of contents on the page. The default is false.
  • navOrder — This number dictates the order in which your routes will render in the sidebar next to their sibling routes.
  • navTitle — Sets the route's title in the sidebar and breadcrumb navigations.
  • showFooter — Indicates whether to display the footer on the page. By default, the footer appears on all pages when you add the footer config to your skyuxconfig.json. To hide the footer on a particular page, set showFooter to false.
  • showInNav — Indicates whether to include the page in the sidebar. The default is true.
