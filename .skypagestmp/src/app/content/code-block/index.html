The code block component allows you to format a block of text as a code snippet with syntax highlighting by wrapping stache-code-block tags around the code block. The code block component is equipped with copy-to-clipboard functionality. When users hover over the code block area, a copy-to-clipboard button appears, and the formatted code copies with the formatting intact. To format shorter code snippets within larger blocks of text, use the code component instead.

If a code block has text within < and > characters, then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, you must use "&lt;" to escape < and "&gt;" to escape >. For example, you must escape the < and > characters in the code snippet public Task<Generic> MethodName() or else the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.

Some Stache features require specific versions of SKY UX Builder, so we strongly recommend updating to the latest version. For example, the code block, copy to clipboard, and sidebar layout components require version 1.19.3 to reference resource files, and version 1.14.0 is required for localization. To update the SKY UX Builder dependency, open the package.json file in your project's root directory and update the version under the devDependencies property.

Code block properties

  • code — (Optional) Passes a code sample into the component. This property binds data, so you can use brackets to evaluate the template expression: [code]. Without the brackets, Angular treats the value as a string and initializes the target property with the string value instead of evaluating the data value.
  • hideCopyToClipboard — (Optional) Indicates whether to hide the copy-to-clipboard button in the code-block header. The default is false.
  • languageType — (Optional) Indicates the language of the code sample. Supports a variety of languages . The default language is markup.
