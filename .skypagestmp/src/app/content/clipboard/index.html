The copy-to-clipboard component creates a button that allows users to copy content to the clipboard. To implement the component, you wrap text in an HTML element and specify an #identifier for that element. Then you place a stache-copy-to-clipboard element on the page and specify the content to copy by referencing its identifier in the copyTarget attribute. When users click the copy-to-clipboard button, the component copies the text to the user's clipboard with formatting intact. If necessary, you can specify different labels for before and after users click the button.

Only HTML elements can use the #identifier notation. If you would like to copy the content of a non-HTML element, such as a Stache component, that element must be wrapped in a div or another valid HTML element.

Some Stache features require specific versions of SKY UX Builder, so we strongly recommend updating to the latest version. For example, the code block, copy to clipboard, and sidebar layout components require version 1.19.3 to reference resource files, and version 1.14.0 is required for localization. To update the SKY UX Builder dependency, open the package.json file in your project's root directory and update the version under the devDependencies property.

Copy-to-clipboard properties

  • copyTarget — Specifies content to copy.

    This property binds data, so you must use brackets to evaluate the template expression: [copyTarget]. Without the brackets, Angular treats the value as a string and initializes the target property with the string value instead of evaluating the data value.

  • buttonText — (Optional) Specifies a label for the copy-to-clipboard button.
  • buttonClickedText — (Optional) Specifies a label to display on the copy-to-clipboard button after users click it.
