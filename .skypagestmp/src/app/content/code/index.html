The code component allows you to format text as a code snippet within a larger block of text. Within the block of text, you wrap stache-code tags around the code snippet. To format longer code snippets that stand alone as separate blocks of text, use the code block component instead.

Code properties

  • escapeCharacters â€” (Optional) Indicates whether to escape special characters such as {{ '{' }} and \. Escaping these characters renders them as text instead of code. Defaults to false.

    If a code snippet has text within < and > characters and you set escapeCharacters="true", then the component automatically appends a closing tag unless the code snippet is valid HTML. To prevent this, you must use "&lt;" to escape < and "&gt;" to escape >. For example, you must escape the < and > characters in the code snippet public Task<Generic> MethodName() or else the component automatically appends </Generic> to the end: public Task<Generic> MethodName()</Generic>.
